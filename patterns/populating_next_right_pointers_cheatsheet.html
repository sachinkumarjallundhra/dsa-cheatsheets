üìå What Is It?

The Populating Next Right Pointers in Each Node problem (LeetCode 116) asks you to connect each node in a perfect binary tree to its next right node. If there is no next right node, the pointer should be set to null.

üéØ Solution Pattern (BFS Level Order)

// Java Solution (BFS)
class Solution {
    public Node connect(Node root) {
        if (root == null) return null;
        Queue<Node> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            Node prev = null;
            for (int i = 0; i < size; i++) {
                Node node = queue.poll();
                if (prev != null) prev.next = node;
                prev = node;
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
        }
        return root;
    }
}

‚úÖ Use Cases

  • Connecting nodes at the same level
  • Tree-based navigation
  • Problems involving next pointers

üìù Algorithm Steps

  1. Initialize a queue and add the root node
  2. For each level, connect nodes using a prev pointer
  3. Add children to the queue for the next level
  4. Repeat until all levels are processed

  • Empty tree (return null)
  • Single node tree

üìå What Is It?

The Binary Tree Level Order Traversal problem (LeetCode 102) asks you to traverse a binary tree level by level from left to right and return the values at each level as a list of lists.

üéØ Solution Pattern (BFS with Queue)

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val);
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            result.add(level);
        }
        return result;
    }
}

‚úÖ Use Cases

  • Level order traversal of trees
  • Finding nodes at a specific depth
  • Problems involving tree layers or breadth

üìù Algorithm Steps

  1. Initialize a queue and add the root node
  2. While the queue is not empty:
    • Record the number of nodes at the current level
    • Process each node at this level, adding their children to the queue
    • Store the values for this level
  3. Repeat until all levels are processed

‚ö†Ô∏è Edge Cases to Consider

  • Empty tree (return empty list)
  • Single node tree
  • Skewed trees (all left or all right)

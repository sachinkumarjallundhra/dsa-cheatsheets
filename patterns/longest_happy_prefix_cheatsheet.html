üìå What Is It?

The Longest Happy Prefix problem (LeetCode 1392) asks you to find the longest prefix of a string which is also a suffix (excluding the whole string itself).

üéØ Solution Pattern (KMP Failure Function)

// Java Solution (KMP Algorithm)
class Solution {
    public String longestPrefix(String s) {
        int n = s.length();
        int[] lps = new int[n];
        int len = 0;
        for (int i = 1; i < n; i++) {
            while (len > 0 && s.charAt(i) != s.charAt(len)) {
                len = lps[len - 1];
            }
            if (s.charAt(i) == s.charAt(len)) {
                len++;
                lps[i] = len;
            }
        }
        return s.substring(0, lps[n - 1]);
    }
}

‚úÖ Use Cases

  • Finding repeated patterns in strings
  • Prefix-suffix matching problems
  • Efficient string processing (KMP algorithm)

üìù Algorithm Steps

  1. Build the LPS (Longest Prefix Suffix) array using KMP preprocessing
  2. The last value in LPS gives the length of the longest happy prefix
  3. Return the substring from 0 to that length

‚è±Ô∏è Time Complexity

  • Time: O(N) where N is the length of the string
  • Space: O(N) for the LPS array

‚ö†Ô∏è Edge Cases to Consider

  • No happy prefix (should return empty string)
  • Whole string is a prefix and suffix (exclude whole string)
  • Single character string

üß™ Example

Input: s = "level"
Output: "l"
Explanation: "l" is the longest prefix which is also a suffix.

Explanation: "abab" is the longest happy prefix.

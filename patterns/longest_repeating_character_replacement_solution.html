Explanation: Replace one 'B' with 'A' â†’ "AABA" or "AAAB"

ğŸ” Approach

  • Use a sliding window [left, right]
  • Track frequency of characters inside the window
  • Keep count of the character with the highest frequency in the window
  • If (window size - maxFreq) > k â†’ shrink the window

ğŸ’» Java Code

public int characterReplacement(String s, int k) {
    int[] freq = new int[26];
    int maxFreq = 0, left = 0, maxLen = 0;

    for (int right = 0; right < s.length(); right++) {
        freq[s.charAt(right) - 'A']++;
        maxFreq = Math.max(maxFreq, freq[s.charAt(right) - 'A']);

        while ((right - left + 1) - maxFreq > k) {
            freq[s.charAt(left) - 'A']--;
            left++;
        }

        maxLen = Math.max(maxLen, right - left + 1);
    }

    return maxLen;
}

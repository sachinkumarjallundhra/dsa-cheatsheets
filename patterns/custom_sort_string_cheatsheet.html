🔤 Custom Sort String – Java Cheat Sheet

📌 What Is It?

The Custom Sort String problem (LeetCode 791) asks you to sort the characters of a string S based on the order defined by another string order. Characters not present in order can appear in any order at the end.

🎯 Solution Pattern (Counting + StringBuilder)

// Java Solution
class Solution {
    public String customSortString(String order, String s) {
        int[] count = new int[26];
        for (char c : s.toCharArray()) count[c - 'a']++;
        StringBuilder sb = new StringBuilder();
        for (char c : order.toCharArray()) {
            while (count[c - 'a']-- > 0) sb.append(c);
        }
        for (char c = 'a'; c <= 'z'; c++) {
            while (count[c - 'a']-- > 0) sb.append(c);
        }
        return sb.toString();
    }
}

✅ Use Cases

  • Sorting strings with custom character priorities
  • Problems involving custom lexicographical order
  • Interview questions on string manipulation

📝 Algorithm Steps

  1. Count occurrences of each character in s
  2. Append characters to result in the order specified by order
  3. Append remaining characters not in order

⏱️ Time Complexity

  • Time: O(N + M) where N is length of s, M is length of order
  • Space: O(1) (fixed alphabet size)

⚠️ Edge Cases to Consider

  • Characters in s not present in order
  • Empty order or s
  • Duplicate characters

🧪 Example

Input: order = "cba", s = "abcd"
Output: "cbad"
Explanation: 'c', 'b', 'a' appear first in order, then 'd' (not in order).

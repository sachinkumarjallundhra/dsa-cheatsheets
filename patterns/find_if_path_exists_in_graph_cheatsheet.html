🔗 Find if Path Exists in Graph – Java Cheat Sheet

📌 What Is It?

The Find if Path Exists in Graph problem (LeetCode 1971) asks you to determine if there is a valid path between two given nodes in an undirected graph.

🎯 Solution Pattern (Union Find / DFS / BFS)

// Java Solution (Union Find)
class Solution {
    public boolean validPath(int n, int[][] edges, int source, int destination) {
        UnionFind uf = new UnionFind(n);
        for (int[] edge : edges) {
            uf.union(edge[0], edge[1]);
        }
        return uf.find(source) == uf.find(destination);
    }
}

class UnionFind {
    int[] parent;
    public UnionFind(int n) {
        parent = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
    }
    public int find(int x) {
        if (parent[x] != x)
            parent[x] = find(parent[x]);
        return parent[x];
    }
    public void union(int x, int y) {
        int px = find(x), py = find(y);
        if (px != py) parent[px] = py;
    }
}

✅ Use Cases

  • Checking connectivity in undirected graphs
  • Union Find applications
  • Graph traversal problems

📝 Algorithm Steps

  1. Initialize Union Find for all nodes
  2. Union all connected nodes via edges
  3. Check if source and destination are in the same component

⏱️ Time Complexity

  • Time: O(E + V) where E is number of edges, V is number of nodes
  • Space: O(V) for Union Find structure

⚠️ Edge Cases to Consider

  • Disconnected graph
  • Source and destination are the same
  • No edges

🧪 Example

Input: n = 3, edges = [[0,1],[1,2],[2,0]], source = 0, destination = 2
Output: true

Input: n = 6, edges = [[0,1],[0,2],[3,5],[5,4],[4,3]], source = 0, destination = 5
Output: false

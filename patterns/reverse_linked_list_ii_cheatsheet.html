📚 Reverse Linked List II – Java Cheat Sheet (One Page)

📌 What Is It?

The Reverse Linked List II problem involves reversing a portion of a linked list between two given positions left and right. The rest of the list remains unchanged.

  • Reverse a sublist of a linked list
  • Handle edge cases like reversing the entire list or a single node
  • Use pointers to manipulate the list in-place

🧱 Pattern Template

public ListNode reverseBetween(ListNode head, int left, int right) {
    if (head == null || left == right) return head;

    // Dummy node to simplify edge cases
    ListNode dummy = new ListNode(0);
    dummy.next = head;

    // Move to the node before the 'left' position
    ListNode prev = dummy;
    for (int i = 1; i < left; i++) {
        prev = prev.next;
    }

    // Reverse the sublist
    ListNode curr = prev.next;
    ListNode next = null;
    for (int i = 0; i < right - left; i++) {
        next = curr.next;
        curr.next = next.next;
        next.next = prev.next;
        prev.next = next;
    }

    return dummy.next;
}

✅ Use Cases

  • Manipulating linked lists in-place
  • Handling edge cases like reversing the entire list or a single node

🧪 Example: Reverse Linked List II

💡 Pro Tips

  • Use a dummy node to simplify edge cases
  • Reverse the sublist in-place using pointers
  • Edge cases: single node, reversing the entire list, or invalid positions

üìå What Is It?

The Minimum Cost to Make at Least One Valid Path in a Grid problem (LeetCode 1368) asks you to find the minimum cost to modify a grid so that there is at least one valid path from the top-left to the bottom-right cell. Each cell has a direction, and you can change the direction at a cost of 1.

üéØ Solution Pattern (0-1 BFS / Dijkstra‚Äôs Algorithm)

class Solution {
    private static final int[][] DIRS = {{0,1},{0,-1},{1,0},{-1,0}};
    public int minCost(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        int[][] cost = new int[m][n];
        for (int[] row : cost) Arrays.fill(row, Integer.MAX_VALUE);
        Deque<int[]> deque = new ArrayDeque<>();
        cost[0][0] = 0;
        deque.offerFirst(new int[]{0, 0});
        while (!deque.isEmpty()) {
            int[] curr = deque.pollFirst();
            int x = curr[0], y = curr[1];
            for (int d = 0; d < 4; d++) {
                int nx = x + DIRS[d][0], ny = y + DIRS[d][1];
                if (nx >= 0 && nx < m && ny >= 0 && ny < n) {
                    int newCost = cost[x][y] + (grid[x][y] == d + 1 ? 0 : 1);
                    if (newCost < cost[nx][ny]) {
                        cost[nx][ny] = newCost;
                        if (grid[x][y] == d + 1)
                            deque.offerFirst(new int[]{nx, ny});
                        else
                            deque.offerLast(new int[]{nx, ny});
                    }
                }
            }
        }
        return cost[m-1][n-1];
    }
}

  • Grid pathfinding with modification cost

üìù Algorithm Steps

  1. Initialize cost matrix with infinity
  2. Use 0-1 BFS (deque) to process cells
  3. Move in the direction of the cell for zero cost, else pay 1
  4. Update cost and add to deque (front for zero cost, back for cost 1)
  5. Continue until bottom-right cell is reached

‚ö†Ô∏è Edge Cases to Consider

  • Single cell grid
  • All directions already valid
  • No valid path unless changes are made
  • Large grids

üß™ Example

Input: grid = [[1,1,3],[3,2,2],[1,1,4]]
Output: 1
Explanation: Change grid[0][2] from 3 to 2 to make a valid path.

Input: grid = [[1,2],[4,3]]
Output: 2
Explanation: Minimum 2 changes needed for a valid path.

üí° Pro Tips

  • Use 0-1 BFS for optimal performance (O(m*n))
  • Deque allows pushing zero-cost moves to front
  • Direction mapping: 1=right, 2=left, 3=down, 4=up
  • Test with grids requiring multiple changes

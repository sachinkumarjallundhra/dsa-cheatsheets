ğŸ“Œ What Is It?

The Jump Game II problem involves finding the minimum number of jumps required to reach the last index of an array. Each element in the array represents the maximum jump length at that position.

ğŸ§± Pattern Template

Greedy Algorithm

class Solution {
    public int jump(int[] nums) {
        int jumps = 0, currentEnd = 0, farthest = 0;

        for (int i = 0; i < nums.length - 1; i++) {
            farthest = Math.max(farthest, i + nums[i]);

            if (i == currentEnd) {
                jumps++;
                currentEnd = farthest;

                if (currentEnd >= nums.length - 1) {
                    break;
                }
            }
        }

        return jumps;
    }
}

ğŸ’¡ Pro Tips

  • Use a greedy approach to always jump to the farthest reachable index.
  • Track the current jump range (`currentEnd`) and update it when the range is exhausted.
  • Break early if the last index is reachable within the current jump.

✂️ Word Break II – Java Cheat Sheet

📌 What Is It?

The Word Break II problem asks you to return all possible sentences where a given string can be segmented into a space-separated sequence of one or more dictionary words.

🧱 Pattern Template (DFS + Memoization)

class Solution {
    public List<String> wordBreak(String s, List<String> wordDict) {
        Set<String> dict = new HashSet<>(wordDict);
        Map<String, List<String>> memo = new HashMap<>();
        return dfs(s, dict, memo);
    }

    private List<String> dfs(String s, Set<String> dict, Map<String, List<String>> memo) {
        if (memo.containsKey(s)) return memo.get(s);
        List<String> res = new ArrayList<>();
        if (s.isEmpty()) {
            res.add("");
            return res;
        }
        for (int end = 1; end <= s.length(); end++) {
            String word = s.substring(0, end);
            if (dict.contains(word)) {
                List<String> sublist = dfs(s.substring(end), dict, memo);
                for (String sub : sublist) {
                    res.add(word + (sub.isEmpty() ? "" : " " + sub));
                }
            }
        }
        memo.put(s, res);
        return res;
    }
}

📊 Time Complexity

  • Time Complexity: Exponential in the worst case, as every substring can be broken in multiple ways.
  • Space Complexity: O(n^2 + k), where n is the string length and k is the total number of characters in all sentences.

✅ Use Cases

  • All possible segmentations of a string using a dictionary.
  • Backtracking with memoization for repeated subproblems.
  • Natural language processing and sentence generation.

🧪 Example

Input: s = "catsanddog", wordDict = ["cat","cats","and","sand","dog"]
Output: ["cats and dog","cat sand dog"]

💡 Pro Tips

  • Use memoization to avoid recomputation for the same substring.
  • Try all possible prefixes and recurse on the suffix.

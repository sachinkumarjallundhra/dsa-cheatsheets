🔁 Linked List Cycle – Java Cheat Sheet

📌 What Is It?

The Linked List Cycle problem (LeetCode 141) asks you to determine if a singly linked list contains a cycle (i.e., a node's next pointer points to a previous node).

🎯 Solution Pattern (Fast and Slow Pointer)

// Java Solution (Floyd's Tortoise and Hare)
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;
    }
    return false;
}

✅ Use Cases

  • Detecting cycles in linked lists
  • Cycle detection in graphs or arrays (with index pointers)
  • Interview questions on pointer manipulation

📝 Algorithm Steps

  1. Initialize two pointers, slow and fast, at the head
  2. Move slow by one step and fast by two steps in each iteration
  3. If slow and fast meet, a cycle exists
  4. If fast reaches the end, there is no cycle

⏱️ Time & Space Complexity

  • Time: O(N) for N nodes
  • Space: O(1) (constant space)

🧪 Example

Input: head = [3,2,0,-4], pos = 1
Output: true

Input: head = [1,2], pos = 0
Output: true

Input: head = [1], pos = -1
Output: false

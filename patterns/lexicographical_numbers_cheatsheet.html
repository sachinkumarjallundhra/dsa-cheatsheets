üìå What Is It?

The Lexicographical Numbers problem (LeetCode 386) asks you to return the numbers from 1 to n in lexicographical (dictionary) order.

üéØ Solution Pattern (DFS Traversal)

// Java Solution (DFS)
class Solution {
    public List<Integer> lexicalOrder(int n) {
        List<Integer> res = new ArrayList<>();
        for (int i = 1; i <= 9; i++) {
            dfs(i, n, res);
        }
        return res;
    }

    private void dfs(int curr, int n, List<Integer> res) {
        if (curr > n) return;
        res.add(curr);
        for (int i = 0; i <= 9; i++) {
            int next = curr * 10 + i;
            if (next > n) break;
            dfs(next, n, res);
        }
    }
}

‚úÖ Use Cases

  • Lexicographical enumeration of numbers
  • Trie-based number traversal
  • Efficient sorting without converting to strings

üìù Algorithm Steps

  1. Start DFS from 1 to 9
  2. For each number, add it to the result list
  3. Recursively visit next numbers by multiplying by 10 and adding 0-9
  4. Stop recursion if the number exceeds n

‚è±Ô∏è Time Complexity

  • Time: O(n) (each number is visited once)
  • Space: O(n) (for the result list and recursion stack)

‚ö†Ô∏è Edge Cases to Consider

  • n is a single digit
  • n = 0 (should return empty list)

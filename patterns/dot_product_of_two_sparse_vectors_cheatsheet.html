üìå What Is It?

The Dot Product of Two Sparse Vectors problem (LeetCode 1570) asks you to efficiently compute the dot product of two sparse vectors. A sparse vector is a vector in which most elements are zero, so storing only non-zero elements saves space and time.

üéØ Solution Pattern (HashMap or Two Pointers)

// Java Solution (HashMap)
class SparseVector {
    Map<Integer, Integer> map = new HashMap<>();
    SparseVector(int[] nums) {
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) map.put(i, nums[i]);
        }
    }
    public int dotProduct(SparseVector vec) {
        int res = 0;
        for (int idx : map.keySet()) {
            if (vec.map.containsKey(idx)) {
                res += map.get(idx) * vec.map.get(idx);
            }
        }
        return res;
    }
}

‚úÖ Use Cases

  • Efficient computation with large, sparse data
  • Machine learning and information retrieval
  • Memory-efficient vector operations

üìù Algorithm Steps

  1. Store non-zero elements and their indices in a HashMap
  2. For dot product, iterate through one map and multiply with matching indices in the other
  3. Sum the products to get the result

‚è±Ô∏è Time Complexity

  • Time: O(L) where L is the number of non-zero elements in the smaller vector
  • Space: O(L) for storing non-zero elements

‚ö†Ô∏è Edge Cases to Consider

  • Vectors of different lengths (problem guarantees same length)
  • All elements are zero (should return 0)
  • One or both vectors are empty

üß™ Example

Input: nums1 = [1,0,0,2,3], nums2 = [0,3,0,4,0]
SparseVector v1 = new SparseVector(nums1);
SparseVector v2 = new SparseVector(nums2);
v1.dotProduct(v2); // Output: 8
Explanation: Only indices 3 and 0 are non-zero, so 2*4 + 1*0 = 8

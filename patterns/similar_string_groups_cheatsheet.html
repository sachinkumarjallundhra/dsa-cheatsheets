🔗 Similar String Groups – Java Cheat Sheet

📌 What Is It?

The Similar String Groups problem (LeetCode 839) asks you to count the number of groups of similar strings. Two strings are similar if you can swap two letters (in different positions) to make them equal, and similarity is transitive. Each string contains only lowercase letters and all strings are of the same length.

🎯 Solution Pattern (Union Find / DSU)

// Java Solution (Union Find)
class Solution {
    public int numSimilarGroups(String[] strs) {
        int n = strs.length;
        UnionFind uf = new UnionFind(n);
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                if (isSimilar(strs[i], strs[j])) {
                    uf.union(i, j);
                }
            }
        }
        return uf.count();
    }

    private boolean isSimilar(String a, String b) {
        int diff = 0;
        for (int i = 0; i < a.length(); i++) {
            if (a.charAt(i) != b.charAt(i)) diff++;
            if (diff > 2) return false;
        }
        return diff == 2 || diff == 0;
    }

    class UnionFind {
        int[] parent;
        int groups;
        public UnionFind(int n) {
            parent = new int[n];
            groups = n;
            for (int i = 0; i < n; i++) parent[i] = i;
        }
        public int find(int x) {
            if (parent[x] != x)
                parent[x] = find(parent[x]);
            return parent[x];
        }
        public void union(int x, int y) {
            int px = find(x), py = find(y);
            if (px != py) {
                parent[px] = py;
                groups--;
            }
        }
        public int count() {
            return groups;
        }
    }
}

✅ Use Cases

  • Grouping similar words or objects
  • Connected components in string graphs
  • Union Find applications

📝 Algorithm Steps

  1. Initialize Union Find for all strings
  2. For each pair of strings, check if they are similar
  3. If similar, union their indices
  4. Return the number of groups

⏱️ Time Complexity

  • Time: O(N^2 * L) where N is number of strings, L is string length
  • Space: O(N) for Union Find structure

⚠️ Edge Cases to Consider

  • All strings are the same
  • No strings are similar
  • Strings with repeated characters

🧪 Example

Input: strs = ["tars","rats","arts","star"]
Output: 2

Input: strs = ["omv","ovm"]
Output: 1

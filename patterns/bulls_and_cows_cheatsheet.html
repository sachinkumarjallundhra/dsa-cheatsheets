The Bulls and Cows problem (LeetCode 299) asks you to compare a secret number and a guess, and return a hint with the number of "bulls" (correct digit and position) and "cows" (correct digit, wrong position).

ğŸ¯ Solution Pattern (HashMap/Array Counting)

// Java Solution
class Solution {
    public String getHint(String secret, String guess) {
        int bulls = 0, cows = 0;
        int[] sCount = new int[10], gCount = new int[10];
        for (int i = 0; i < secret.length(); i++) {
            if (secret.charAt(i) == guess.charAt(i)) {
                bulls++;
            } else {
                sCount[secret.charAt(i) - '0']++;
                gCount[guess.charAt(i) - '0']++;
            }
        }
        for (int i = 0; i < 10; i++) {
            cows += Math.min(sCount[i], gCount[i]);
        }
        return bulls + "A" + cows + "B";
    }
}

ğŸ“ Algorithm Steps

  1. Iterate through both strings, count bulls (exact matches)
  2. For non-bull digits, count their occurrences in secret and guess
  3. Sum the minimum counts for each digit to get cows
  4. Return the result in "xAyB" format

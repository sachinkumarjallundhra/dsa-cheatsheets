🌳 Maximum Depth of Binary Tree – Java Cheat Sheet

📌 What Is It?

The Maximum Depth of Binary Tree problem (LeetCode 104) asks you to find the maximum depth (height) of a binary tree. The depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

🎯 Solution Pattern (Recursive DFS)

class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;
        int left = maxDepth(root.left);
        int right = maxDepth(root.right);
        return 1 + Math.max(left, right);
    }
}

✅ Use Cases

  • Tree height calculation
  • Balanced tree checks
  • Recursive tree traversal practice

📝 Algorithm Steps

  1. If the node is null, return 0
  2. Recursively find the depth of left and right subtrees
  3. Return 1 + max(left, right)

⚠️ Edge Cases to Consider

  • Empty tree (depth is 0)
  • Single node tree (depth is 1)

🧪 Example

Input: root = [3,9,20,null,null,15,7]
Output: 3
Explanation: The maximum depth is 3 (path: 3 → 20 → 15 or 3 → 20 → 7).

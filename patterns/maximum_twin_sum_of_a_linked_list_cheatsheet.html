🔗 Maximum Twin Sum of a Linked List – Java Cheat Sheet

📌 What Is It?

The Maximum Twin Sum of a Linked List problem (LeetCode 2130) asks you to find the maximum twin sum in a singly linked list. A twin sum is defined as the sum of the ith node from the start and the ith node from the end (for all i in the first half of the list).

🎯 Solution Pattern (Find Middle, Reverse Second Half, Compare)

// Java Solution
public int pairSum(ListNode head) {
    // 1. Find the middle of the list
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    // 2. Reverse the second half
    ListNode prev = null, curr = slow;
    while (curr != null) {
        ListNode nextTemp = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextTemp;
    }
    // 3. Calculate twin sums
    int maxSum = 0;
    ListNode first = head, second = prev;
    while (second != null) {
        maxSum = Math.max(maxSum, first.val + second.val);
        first = first.next;
        second = second.next;
    }
    return maxSum;
}

✅ Use Cases

  • Finding maximum twin sum in a linked list
  • Interview questions on pointer manipulation and list reversal
  • Splitting and merging linked lists

📝 Algorithm Steps

  1. Use fast and slow pointers to find the middle of the list
  2. Reverse the second half of the list
  3. Iterate through both halves, calculating twin sums
  4. Return the maximum twin sum found

⏱️ Time & Space Complexity

  • Time: O(N) for N nodes
  • Space: O(1) (in-place reversal)

🧪 Example

Input: head = [5,4,2,1]
Output: 6
Explanation: Twin sums are 5+1=6 and 4+2=6. Maximum is 6.

Input: head = [1,100000]
Output: 100001

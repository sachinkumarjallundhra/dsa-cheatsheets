The Find the First K Missing Positive Numbers problem asks you to find the first k smallest missing positive integers from an unsorted array. The array may contain duplicates and negative numbers.

ğŸ§± Pattern Template (Cyclic Sort Solution)

class Solution {
    public List<Integer> findKMissingPositive(int[] nums, int k) {
        int n = nums.length;
        int i = 0;
        while (i < n) {
            if (nums[i] > 0 && nums[i] <= n && nums[nums[i] - 1] != nums[i]) {
                int temp = nums[i];
                nums[i] = nums[temp - 1];
                nums[temp - 1] = temp;
            } else {
                i++;
            }
        }
        List<Integer> result = new ArrayList<>();
        Set<Integer> extra = new HashSet<>();
        for (i = 0; i < n && result.size() < k; i++) {
            if (nums[i] != i + 1) {
                result.add(i + 1);
                extra.add(nums[i]);
            }
        }
        int next = n + 1;
        while (result.size() < k) {
            if (!extra.contains(next)) result.add(next);
            next++;
        }
        return result;
    }
}

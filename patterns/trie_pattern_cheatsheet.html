üìå What Is It?

The Trie (Prefix Tree) is a tree-like data structure used to efficiently store and retrieve keys in a dataset of strings. It is commonly used for autocomplete, spell checking, and prefix-based queries.

üéØ Solution Pattern (Basic Trie Implementation)

// Java Trie Implementation
class TrieNode {
    TrieNode[] children = new TrieNode[26];
    boolean isEnd = false;
}

class Trie {
    TrieNode root = new TrieNode();

    public void insert(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int idx = c - 'a';
            if (node.children[idx] == null) node.children[idx] = new TrieNode();
            node = node.children[idx];
        }
        node.isEnd = true;
    }

    public boolean search(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int idx = c - 'a';
            if (node.children[idx] == null) return false;
            node = node.children[idx];
        }
        return node.isEnd;
    }

    public boolean startsWith(String prefix) {
        TrieNode node = root;
        for (char c : prefix.toCharArray()) {
            int idx = c - 'a';
            if (node.children[idx] == null) return false;
            node = node.children[idx];
        }
        return true;
    }
}

‚úÖ Use Cases

  • Autocomplete and prefix search
  • Spell checker
  • Word dictionary problems
  • Efficient string matching

üìù Algorithm Steps

  1. Initialize the root node
  2. For insert, traverse or create nodes for each character
  3. For search, traverse nodes and check isEnd
  4. For startsWith, traverse nodes for the prefix

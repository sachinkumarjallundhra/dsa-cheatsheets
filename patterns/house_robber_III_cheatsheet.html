üìå What Is It?

The House Robber III problem involves finding the maximum amount of money a thief can rob from houses arranged in a binary tree. The constraint is that no two directly connected houses can be robbed.

üß± Pattern Template

class Solution {
    public int rob(TreeNode root) {
        int[] result = robHelper(root);
        return Math.max(result[0], result[1]);
    }

    private int[] robHelper(TreeNode node) {
        if (node == null) {
            return new int[]{0, 0}; // {max if not robbed, max if robbed}
        }

        int[] left = robHelper(node.left);
        int[] right = robHelper(node.right);

        // If the current node is not robbed, take the max of both children
        int notRobbed = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);

        // If the current node is robbed, add its value to the "not robbed" values of children
        int robbed = node.val + left[0] + right[0];

        return new int[]{notRobbed, robbed};
    }
}

üìä Time Complexity

  • Time Complexity: O(N), where N is the number of nodes in the tree. Each node is visited once.
  • Space Complexity: O(H), where H is the height of the tree, for the recursion stack.

üß™ Example

Input: root = [3,2,3,null,3,null,1]
Output: 7

Explanation:
- Rob house 3 (root) and houses 3 and 1 (children of the right subtree).
- Maximum amount = 3 + 3 + 1 = 7.

üí° Pro Tips

  • Use a helper function that returns two values: max money if the node is robbed and max money if it is not robbed.
  • Prune unnecessary branches early by returning {0, 0} for null nodes.
  • Edge cases: Empty tree, single-node tree.

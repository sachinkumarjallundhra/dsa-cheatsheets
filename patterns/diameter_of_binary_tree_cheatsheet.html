🌳 Diameter of Binary Tree – Java Cheat Sheet

📌 What Is It?

The Diameter of Binary Tree problem (LeetCode 543) asks you to find the length of the longest path between any two nodes in a binary tree. The path may or may not pass through the root.

🎯 Solution Pattern (DFS for Height & Diameter)

class Solution {
    private int diameter = 0;
    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);
        return diameter;
    }
    private int dfs(TreeNode node) {
        if (node == null) return 0;
        int left = dfs(node.left);
        int right = dfs(node.right);
        diameter = Math.max(diameter, left + right);
        return 1 + Math.max(left, right);
    }
}

✅ Use Cases

  • Finding longest path in trees
  • Network latency analysis
  • Tree structure optimization

📝 Algorithm Steps

  1. Use DFS to compute the height of left and right subtrees
  2. At each node, update diameter as left height + right height
  3. Return the maximum diameter found

⚠️ Edge Cases to Consider

  • Empty tree (diameter is 0)
  • Single node tree (diameter is 0)
  • Skewed trees (all left or all right)

🧪 Example

Input: root = [1,2,3,4,5]
Output: 3
Explanation: The longest path is 4→2→1→3 or 5→2→1→3.

Input: root = [1,2]
Output: 1
Explanation: The longest path is 2→1.

💡 Pro Tips

  • Diameter is the number of edges, not nodes
  • DFS is optimal for subtree calculations
  • Update diameter at every node

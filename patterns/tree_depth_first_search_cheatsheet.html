üìå What Is It?

Tree Depth-First Search (DFS) is a traversal technique for trees where you explore as far as possible along each branch before backtracking. DFS is used for subtree calculations, path finding, and many recursive tree problems.

üéØ Solution Pattern (Recursive DFS)

// Binary Tree DFS (Preorder)
void dfs(TreeNode node) {
    if (node == null) return;
    // Process node value
    dfs(node.left);
    dfs(node.right);
}

// DFS with return value (e.g., subtree sum)
int dfs(TreeNode node) {
    if (node == null) return 0;
    int left = dfs(node.left);
    int right = dfs(node.right);
    // Combine results
    return node.val + left + right;
}

‚úÖ Use Cases

  • Subtree sum, size, or depth calculation
  • Finding paths or ancestors
  • Lowest common ancestor (LCA)
  • Tree serialization/deserialization
  • Checking tree properties (balanced, symmetric, etc.)

üìù Algorithm Steps

  1. Start DFS from the root node
  2. Process the current node (preorder, inorder, or postorder)
  3. Recursively visit left and right children
  4. Combine results as needed for the problem

‚ö†Ô∏è Edge Cases to Consider

  • Empty tree (root is null)
  • Single node tree
  • Skewed trees (all left or all right)
  • Large/deep trees (stack overflow risk)

üß™ Example

// Example: Sum of all nodes in a binary tree
int sum(TreeNode root) {
    if (root == null) return 0;
    return root.val + sum(root.left) + sum(root.right);
}

üí° Pro Tips

  • Use recursion for most tree DFS problems
  • For iterative DFS, use a stack
  • Preorder: process before children; Postorder: process after children
  • Return values from DFS for subtree calculations
  • Careful with null checks to avoid NullPointerException

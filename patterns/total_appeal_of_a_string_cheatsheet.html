ğŸ“Œ What Is It?

The Total Appeal of a String problem (LeetCode 2262) asks you to find the sum of the appeal of all substrings of a string. The appeal of a substring is the number of distinct characters in it.

ğŸ¯ Solution Pattern (Last Occurrence Tracking)

// Java Solution
class Solution {
    public long appealSum(String s) {
        long res = 0, cur = 0;
        int[] last = new int[26];
        Arrays.fill(last, -1);
        for (int i = 0; i < s.length(); i++) {
            cur += i - last[s.charAt(i) - 'a'];
            res += cur;
            last[s.charAt(i) - 'a'] = i;
        }
        return res;
    }
}

  1. Initialize an array to track the last occurrence of each character
  2. For each character, update the current appeal by adding the distance from its last occurrence
  3. Add the current appeal to the result
  4. Update the last occurrence of the character

❓ Missing Number – Java Cheat Sheet

📌 What Is It?

The Missing Number problem asks you to find the single missing number from an array containing n distinct numbers taken from 0, 1, 2, ..., n.

🧱 Pattern Template (Cyclic Sort Solution)

class Solution {
    public int missingNumber(int[] nums) {
        int i = 0;
        while (i < nums.length) {
            if (nums[i] < nums.length && nums[i] != nums[nums[i]]) {
                int temp = nums[i];
                nums[i] = nums[temp];
                nums[temp] = temp;
            } else {
                i++;
            }
        }
        for (i = 0; i < nums.length; i++) {
            if (nums[i] != i) return i;
        }
        return nums.length;
    }
}

🧱 Pattern Template (Sum Formula Solution)

class Solution {
    public int missingNumber(int[] nums) {
        int n = nums.length;
        int sum = n * (n + 1) / 2;
        for (int num : nums) sum -= num;
        return sum;
    }
}

📊 Time Complexity

  • Time Complexity: O(n)
  • Space Complexity: O(1)

✅ Use Cases

  • Finding a missing element in a sequence
  • Data validation and integrity checks
  • Problems with a known range and one missing value

🧪 Example

Input: nums = [3,0,1]
Output: 2

Input: nums = [0,1]
Output: 2

Input: nums = [9,6,4,2,3,5,7,0,1]
Output: 8

💡 Pro Tips

  • Sum formula is fastest and simplest for this problem.
  • Cyclic sort is useful for in-place rearrangement and related problems.
  • Can also solve with XOR for O(1) space and O(n) time.

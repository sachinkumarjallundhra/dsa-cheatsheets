üìå What Is It?

The Word Ladder problem (LeetCode 127) asks you to find the shortest transformation sequence from a start word to an end word, changing one letter at a time, with each intermediate word existing in a given word list.

üéØ Solution Pattern (BFS + Preprocessing)

// Java Solution
class Solution {
    public int ladderLength(String beginWord, String endWord, List<String> wordList) {
        Set<String> wordSet = new HashSet<>(wordList);
        if (!wordSet.contains(endWord)) return 0;
        Queue<String> queue = new LinkedList<>();
        queue.offer(beginWord);
        Set<String> visited = new HashSet<>();
        visited.add(beginWord);
        int steps = 1;
        while (!queue.isEmpty()) {
            int size = queue.size();
            for (int i = 0; i < size; i++) {
                String word = queue.poll();
                if (word.equals(endWord)) return steps;
                char[] arr = word.toCharArray();
                for (int j = 0; j < arr.length; j++) {
                    char old = arr[j];
                    for (char c = 'a'; c <= 'z'; c++) {
                        arr[j] = c;
                        String next = new String(arr);
                        if (wordSet.contains(next) && !visited.contains(next)) {
                            queue.offer(next);
                            visited.add(next);
                        }
                    }
                    arr[j] = old;
                }
            }
            steps++;
        }
        return 0;
    }
}

How it works: Use BFS to explore all possible one-letter transformations. Each level of BFS represents a transformation step. Stop when the end word is found.

üìù Algorithm Steps

  1. Add all words to a set for O(1) lookup
  2. Use BFS starting from beginWord
  3. For each word, try all possible one-letter changes
  4. If the new word is in the set and not visited, add to queue
  5. Return steps when endWord is reached

‚ö†Ô∏è Edge Cases to Consider

  • endWord not in word list (return 0)
  • No possible transformation (return 0)
  • Words of different lengths (invalid input)

ğŸ“Œ What Is It?

The Number of Valid Subarrays problem (LeetCode 1063) asks you to count the number of contiguous subarrays where every element in the subarray is greater than or equal to the last element of the subarray.

ğŸ§± Pattern Template (Monotonic Stack)

class Solution {
    public int validSubarrays(int[] nums) {
        int n = nums.length, res = 0;
        Stack<Integer> stack = new Stack<>();
        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && nums[i] < nums[stack.peek()]) {
                int idx = stack.pop();
                res += i - idx;
            }
            stack.push(i);
        }
        while (!stack.isEmpty()) {
            int idx = stack.pop();
            res += n - idx;
        }
        return res;
    }
}

  • For each element, pop from the stack until the stack top is less than or equal to the current element.
  • After the main loop, process any remaining indices in the stack.

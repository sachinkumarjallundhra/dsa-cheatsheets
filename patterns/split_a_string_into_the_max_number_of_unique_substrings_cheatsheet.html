🧩 Split a String Into the Max Number of Unique Substrings – Java Cheat Sheet

📌 What Is It?

The Split a String Into the Max Number of Unique Substrings problem asks you to split a string into the maximum number of unique non-empty substrings. Each substring must be unique in the split.

🧱 Pattern Template

class Solution {
    public int maxUniqueSplit(String s) {
        return backtrack(s, 0, new HashSet<>());
    }

    private int backtrack(String s, int start, Set<String> seen) {
        if (start == s.length()) return seen.size();
        int max = 0;
        for (int end = start + 1; end <= s.length(); end++) {
            String sub = s.substring(start, end);
            if (!seen.contains(sub)) {
                seen.add(sub);
                max = Math.max(max, backtrack(s, end, seen));
                seen.remove(sub); // backtrack
            }
        }
        return max;
    }
}

📊 Time Complexity

  • Time Complexity: O(N!), where N is the length of the string (due to all possible splits and uniqueness checks).
  • Space Complexity: O(N), for the recursion stack and the set of seen substrings.

🧪 Example

Input: s = "ababccc"
Output: 5

Explanation:
- One possible split: "a", "b", "ab", "c", "cc".
- All substrings are unique, and the split count is maximized.

💡 Pro Tips

  • Use a HashSet to track unique substrings in the current split.
  • Backtrack by removing the substring from the set after exploring a branch.
  • Edge cases: All characters are the same, or all are unique.

The Counting Bits problem asks you to return an array ans where ans[i] is the number of 1's (set bits) in the binary representation of i for every i in the range [0, n].

ğŸ§± Pattern Template (Dynamic Programming)

class Solution {
    public int[] countBits(int n) {
        int[] ans = new int[n + 1];
        for (int i = 1; i <= n; i++) {
            ans[i] = ans[i >> 1] + (i & 1);
        }
        return ans;
    }
}

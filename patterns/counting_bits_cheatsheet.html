🧮 Counting Bits – Java Cheat Sheet

📌 What Is It?

The Counting Bits problem asks you to return an array ans where ans[i] is the number of 1's (set bits) in the binary representation of i for every i in the range [0, n].

🧱 Pattern Template (Dynamic Programming)

class Solution {
    public int[] countBits(int n) {
        int[] ans = new int[n + 1];
        for (int i = 1; i <= n; i++) {
            ans[i] = ans[i >> 1] + (i & 1);
        }
        return ans;
    }
}

📊 Time Complexity

  • Time Complexity: O(n)
  • Space Complexity: O(n)

✅ Use Cases

  • Bit manipulation and binary representation problems.
  • Dynamic programming with subproblem reuse.
  • Efficiently counting set bits for a range of numbers.

📘 Common LeetCode Problems

🧪 Example

Input: n = 5
Output: [0,1,1,2,1,2]

Explanation:
- 0: 0b0 → 0
- 1: 0b1 → 1
- 2: 0b10 → 1
- 3: 0b11 → 2
- 4: 0b100 → 1
- 5: 0b101 → 2

💡 Pro Tips

  • Use ans[i] = ans[i >> 1] + (i & 1) for efficient computation.
  • Bitwise operations are fast and avoid repeated computation.
  • Great for practicing bit manipulation and DP together.

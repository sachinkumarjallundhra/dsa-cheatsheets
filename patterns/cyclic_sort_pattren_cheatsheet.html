🔄 Cyclic Sort Pattern – Java Cheat Sheet

📌 What Is It?

The Cyclic Sort pattern is used to efficiently place numbers in their correct indices in an array containing numbers in a known range (often 1 to n or 0 to n-1). It is especially useful for problems involving finding missing or duplicate numbers.

🧱 Pattern Template

class Solution {
    public void cyclicSort(int[] nums) {
        int i = 0;
        while (i < nums.length) {
            int correct = nums[i] - 1; // for 1 to n, use nums[i] - 1; for 0 to n-1, use nums[i]
            if (nums[i] >= 1 && nums[i] <= nums.length && nums[i] != nums[correct]) {
                int temp = nums[i];
                nums[i] = nums[correct];
                nums[correct] = temp;
            } else {
                i++;
            }
        }
    }
}

📊 Time Complexity

  • Time Complexity: O(n)
  • Space Complexity: O(1)

✅ Use Cases

  • Finding missing or duplicate numbers in an array
  • Sorting numbers in a known range in-place
  • Problems where each value should be at a specific index

🧪 Example

Input: nums = [3, 1, 5, 4, 2]
Output: [1, 2, 3, 4, 5]

Input: nums = [4, 3, 2, 7, 8, 2, 3, 1]
Output: [1, 2, 3, 4, 7, 8, 2, 3] (sorted in-place for 1 to n)

💡 Pro Tips

  • Use when numbers are in a known range and you want O(n) time and O(1) space.
  • Great for finding missing/duplicate numbers without extra space.
  • Be careful with index calculation for 0-based vs 1-based arrays.

🤖 Does your problem match this pattern?

Yes, if either of these conditions are fulfilled:

  • Limited range integer arrays: The problem involves an input array of integers in a small range, usually [1−n].
  • Finding missing or duplicate elements: The problem requires us to identify missing or duplicate elements in an array.
No, if any of these conditions is fulfilled:
  • Non-integer values: The input array contains non-integer values.
  • Non-array format: The input data is not originally in an array, nor can it be mapped to an array.
  • Stability requirement: The problem requires stable sorting.

🌍 Real-World Applications

  • Computational biology: Finding the kth missing gene in a DNA sequence where genes are numbered 1...n.
  • Playing card sorting: Sorting a shuffled deck of cards (represented as integers [1-52]) efficiently.
  • Data validation: Quickly identifying missing or duplicate values in datasets expected to have distinct values within a certain range.
  • Package delivery routing: Mapping delivery addresses to integers and using cyclic sort to optimize delivery routes.

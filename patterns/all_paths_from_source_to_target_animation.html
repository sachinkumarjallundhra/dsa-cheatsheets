🛣️ All Paths From Source to Target – Java Cheat Sheet

📌 What Is It?

The All Paths From Source to Target problem asks you to find all possible paths from node 0 to node n-1 in a directed acyclic graph (DAG) represented as an adjacency list.

🧱 Pattern Template

class Solution {
    public List<List<Integer>> allPathsSourceTarget(int[][] graph) {
        List<List<Integer>> result = new ArrayList<>();
        List<Integer> path = new ArrayList<>();
        path.add(0);
        dfs(graph, 0, path, result);
        return result;
    }

    private void dfs(int[][] graph, int node, List<Integer> path, List<List<Integer>> result) {
        if (node == graph.length - 1) {
            result.add(new ArrayList<>(path));
            return;
        }
        for (int next : graph[node]) {
            path.add(next);
            dfs(graph, next, path, result);
            path.remove(path.size() - 1); // backtrack
        }
    }
}

📊 Time Complexity

  • Time Complexity: O(2^N * N), where N is the number of nodes (since there can be up to 2^N paths in the worst case).
  • Space Complexity: O(2^N * N), for storing all paths and the recursion stack.

✅ Use Cases

  • Finding all possible paths in a DAG.
  • Backtracking and path enumeration in graphs.
  • Problems involving traversal from source to target.

📘 Common LeetCode Problems

  • 797. All Paths From Source to Target [Medium] Backtracking Graph

🧪 Example

Input: graph = [[1,2],[3],[3],[]]
Output: [[0,1,3],[0,2,3]]

Explanation:
- There are two paths from node 0 to node 3: 0→1→3 and 0→2→3.

💡 Pro Tips

  • Use DFS with backtracking to explore all possible paths.
  • Since the graph is a DAG, you don't need to worry about cycles.
  • Edge cases: No path exists, or only one path exists.

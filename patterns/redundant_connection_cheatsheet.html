The Redundant Connection problem (LeetCode 684) asks you to find an edge that can be removed from a graph so that the graph remains a tree (no cycles). The graph is given as a list of edges forming a connected undirected graph with one extra edge.

ğŸ¯ Solution Pattern (Union Find / DSU)

// Java Solution (Union Find)
class Solution {
    public int[] findRedundantConnection(int[][] edges) {
        int n = edges.length;
        int[] parent = new int[n + 1];
        for (int i = 1; i <= n; i++) parent[i] = i;
        for (int[] edge : edges) {
            int u = edge[0], v = edge[1];
            int pu = find(parent, u), pv = find(parent, v);
            if (pu == pv) return edge;
            parent[pu] = pv;
        }
        return new int[0];
    }
    private int find(int[] parent, int x) {
        if (parent[x] != x) parent[x] = find(parent, parent[x]);
        return parent[x];
    }
}

ğŸ“ Algorithm Steps

  1. Initialize parent array for Union Find
  2. For each edge, check if endpoints are already connected
  3. If yes, this edge is redundant (forms a cycle)
  4. Otherwise, union the endpoints

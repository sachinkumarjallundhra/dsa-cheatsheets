✖️ Maximum Product Subarray – Java Cheat Sheet

📌 What Is It?

The Maximum Product Subarray problem asks you to find the contiguous subarray within an array (containing at least one number) which has the largest product.

🧱 Pattern Template (Dynamic Programming)

class Solution {
    public int maxProduct(int[] nums) {
        int maxProd = nums[0], minProd = nums[0], result = nums[0];
        for (int i = 1; i < nums.length; i++) {
            int tempMax = Math.max(nums[i], Math.max(maxProd * nums[i], minProd * nums[i]));
            minProd = Math.min(nums[i], Math.min(maxProd * nums[i], minProd * nums[i]));
            maxProd = tempMax;
            result = Math.max(result, maxProd);
        }
        return result;
    }
}

📊 Time Complexity

  • Time Complexity: O(N), where N is the length of the array.
  • Space Complexity: O(1), constant extra space.

✅ Use Cases

  • Subarray product problems.
  • Handling negative numbers in products.
  • Dynamic programming with multiple states (max/min).

📘 Common LeetCode Problems

  • 152. Maximum Product Subarray [Medium] DP

🧪 Example

Input: nums = [2,3,-2,4]
Output: 6

Explanation:
- The subarray [2,3] has the largest product 6.

💡 Pro Tips

  • Track both max and min products at each step (because of negatives).
  • Reset on encountering zero if needed.
  • Edge cases: single element, all negatives, zeros in array.

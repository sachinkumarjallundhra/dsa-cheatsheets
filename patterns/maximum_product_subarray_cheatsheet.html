The Maximum Product Subarray problem asks you to find the contiguous subarray within an array (containing at least one number) which has the largest product.

ğŸ§± Pattern Template (Dynamic Programming)

class Solution {
    public int maxProduct(int[] nums) {
        int maxProd = nums[0], minProd = nums[0], result = nums[0];
        for (int i = 1; i < nums.length; i++) {
            int tempMax = Math.max(nums[i], Math.max(maxProd * nums[i], minProd * nums[i]));
            minProd = Math.min(nums[i], Math.min(maxProd * nums[i], minProd * nums[i]));
            maxProd = tempMax;
            result = Math.max(result, maxProd);
        }
        return result;
    }
}

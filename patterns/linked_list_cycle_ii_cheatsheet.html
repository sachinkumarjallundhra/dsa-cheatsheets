🔁 Linked List Cycle II – Java Cheat Sheet

📌 What Is It?

The Linked List Cycle II problem (LeetCode 142) asks you to find the node where the cycle begins in a singly linked list. If there is no cycle, return null.

🎯 Solution Pattern (Floyd's Tortoise and Hare)

// Java Solution (Floyd's Tortoise and Hare)
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    // First, determine if a cycle exists
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) {
            // Cycle detected, find the entry point
            ListNode entry = head;
            while (entry != slow) {
                entry = entry.next;
                slow = slow.next;
            }
            return entry;
        }
    }
    return null;
}

✅ Use Cases

  • Finding the start of a cycle in a linked list
  • Cycle detection and entry point finding
  • Interview questions on pointer manipulation

📝 Algorithm Steps

  1. Use fast and slow pointers to detect a cycle
  2. If a cycle is detected, initialize a pointer at the head
  3. Move both pointers one step at a time; the meeting point is the cycle start
  4. If no cycle, return null

⏱️ Time & Space Complexity

  • Time: O(N) for N nodes
  • Space: O(1) (constant space)

🧪 Example

Input: head = [3,2,0,-4], pos = 1
Output: Reference to node with value 2

Input: head = [1,2], pos = 0
Output: Reference to node with value 1

Input: head = [1], pos = -1
Output: null

üìå What Is It?

The Longest Common Prefix problem (LeetCode 14) asks you to find the longest common prefix string amongst an array of strings. If there is no common prefix, return an empty string.

üéØ Solution Pattern (Trie Approach)

// Java Solution (Trie)
class TrieNode {
    TrieNode[] children = new TrieNode[26];
    int count = 0;
}

class Solution {
    public String longestCommonPrefix(String[] strs) {
        if (strs == null || strs.length == 0) return "";
        TrieNode root = new TrieNode();
        int n = strs.length;

        // Build Trie
        for (String word : strs) {
            TrieNode node = root;
            for (char c : word.toCharArray()) {
                int idx = c - 'a';
                if (node.children[idx] == null) node.children[idx] = new TrieNode();
                node = node.children[idx];
                node.count++;
            }
        }

        // Find prefix
        StringBuilder prefix = new StringBuilder();
        TrieNode node = root;
        outer:
        for (char c : strs[0].toCharArray()) {
            int idx = c - 'a';
            if (node.children[idx] == null || node.children[idx].count != n) break;
            prefix.append(c);
            node = node.children[idx];
        }
        return prefix.toString();
    }
}

üéØ Solution Pattern (Horizontal Scanning)

// Java Solution (Horizontal Scanning)
class Solution {
    public String longestCommonPrefix(String[] strs) {
        if (strs == null || strs.length == 0) return "";
        String prefix = strs[0];
        for (int i = 1; i < strs.length; i++) {
            while (strs[i].indexOf(prefix) != 0) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) return "";
            }
        }
        return prefix;
    }
}

üìù Algorithm Steps

  1. Start with the first string as the prefix
  2. Iterate through the rest of the strings
  3. Reduce the prefix until it matches the start of each string
  4. If prefix becomes empty, return ""

‚è±Ô∏è Time Complexity

  • Time: O(S), where S is the sum of all characters in all strings
  • Space: O(1) (ignoring output)

‚ö†Ô∏è Edge Cases to Consider

  • Empty input array
  • Array contains empty string(s)
  • No common prefix

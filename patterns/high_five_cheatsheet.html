ğŸ“Œ What Is It?

The High Five problem (LeetCode 1086) asks you to calculate the average of each student's top five scores from a list of [student_id, score] pairs.

ğŸ¯ Solution Pattern (HashMap + PriorityQueue)

// Java Solution
class Solution {
    public int[][] highFive(int[][] items) {
        Map<Integer, PriorityQueue<Integer>> map = new HashMap<>();
        for (int[] item : items) {
            int id = item[0], score = item[1];
            map.computeIfAbsent(id, k -> new PriorityQueue<>());
            map.get(id).offer(score);
            if (map.get(id).size() > 5) map.get(id).poll();
        }
        int[][] res = new int[map.size()][2];
        int i = 0;
        for (int id : map.keySet()) {
            int sum = 0;
            for (int score : map.get(id)) sum += score;
            res[i++] = new int[]{id, sum / 5};
        }
        Arrays.sort(res, Comparator.comparingInt(a -> a[0]));
        return res;
    }
}

ğŸ“ Algorithm Steps

  1. Use a HashMap to group scores by student ID
  2. For each student, use a min-heap (PriorityQueue) to keep their top 5 scores
  3. Sum the top 5 scores and calculate the average
  4. Sort the result by student ID

â±ï¸ Time Complexity

  • Time: O(N log 5 + S log S) where N is number of items, S is number of students
  • Space: O(S * 5) for storing scores

âš ï¸ Edge Cases to Consider

  • Students with fewer than 5 scores (problem guarantees at least 5)
  • Duplicate scores
  • Large number of students

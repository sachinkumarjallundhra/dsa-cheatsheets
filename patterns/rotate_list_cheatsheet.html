The Rotate List problem (LeetCode 61) asks you to rotate a singly linked list to the right by k places.

ğŸ¯ Solution Pattern (Two Pointers + Modulo)

// Java Solution
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        if (head == null || head.next == null || k == 0) return head;
        // Compute the length and connect tail to head
        ListNode oldTail = head;
        int n = 1;
        while (oldTail.next != null) {
            oldTail = oldTail.next;
            n++;
        }
        oldTail.next = head; // Make it circular

        // Find new tail: (n - k % n - 1)th node, new head: (n - k % n)th node
        int stepsToNewHead = n - k % n;
        ListNode newTail = head;
        for (int i = 1; i < stepsToNewHead; i++) {
            newTail = newTail.next;
        }
        ListNode newHead = newTail.next;
        newTail.next = null;
        return newHead;
    }
}

  1. Find the length of the list and connect the tail to the head to form a cycle
  2. Calculate the new tail's position: (n - k % n - 1)
  3. Break the cycle at the new tail and return the new head

🌡️ Daily Temperatures – Java Cheat Sheet

📌 What Is It?

The Daily Temperatures problem (LeetCode 739) asks you to find, for each day, how many days you would have to wait until a warmer temperature. If there is no future day for which this is possible, put 0 instead.

🧱 Pattern Template (Monotonic Stack)

class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        int n = temperatures.length;
        int[] answer = new int[n];
        Stack<Integer> stack = new Stack<>(); // stores indices

        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
                int prevIndex = stack.pop();
                answer[prevIndex] = i - prevIndex;
            }
            stack.push(i);
        }
        return answer;
    }
}

  • Monotonic stack applications

💡 Pro Tips

  • Use a stack to keep track of indices with unresolved warmer days.
  • Pop from the stack when you find a warmer temperature and update the answer.
  • Monotonic stacks are useful for "next greater" type problems.

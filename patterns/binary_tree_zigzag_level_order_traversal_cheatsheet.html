üìå What Is It?

The Binary Tree Zigzag Level Order Traversal problem (LeetCode 103) asks you to traverse a binary tree level by level, but alternate the direction at each level (left-to-right, then right-to-left).

üéØ Solution Pattern (BFS with Direction Toggle)

class Solution {
    public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        boolean leftToRight = true;
        while (!queue.isEmpty()) {
            int size = queue.size();
            LinkedList<Integer> level = new LinkedList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                if (leftToRight) {
                    level.addLast(node.val);
                } else {
                    level.addFirst(node.val);
                }
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            result.add(level);
            leftToRight = !leftToRight;
        }
        return result;
    }
}

‚úÖ Use Cases

  • Zigzag/spiral traversal of trees
  • Problems requiring alternate direction at each level
  • Visualizing tree layers in a zigzag manner

üìù Algorithm Steps

  1. Initialize a queue and add the root node
  2. Use a boolean to track direction (left-to-right or right-to-left)
  3. While the queue is not empty:
    • Process each node at the current level
    • Add values to the level list at the appropriate end
    • Add children to the queue
  4. Toggle direction after each level

‚ö†Ô∏è Edge Cases to Consider

  • Empty tree (return empty list)
  • Single node tree
  • Skewed trees (all left or all right)

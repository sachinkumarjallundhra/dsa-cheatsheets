✂️ Word Break – Java Cheat Sheet

📌 What Is It?

The Word Break problem asks if a given string can be segmented into a space-separated sequence of one or more dictionary words.

🧱 Pattern Template (DP Solution)

class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> dict = new HashSet<>(wordDict);
        boolean[] dp = new boolean[s.length() + 1];
        dp[0] = true;
        for (int i = 1; i <= s.length(); i++) {
            for (int j = 0; j < i; j++) {
                if (dp[j] && dict.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }
        return dp[s.length()];
    }
}

📊 Time Complexity

  • Time Complexity: O(n^3), where n is the length of the string (due to substring and nested loops).
  • Space Complexity: O(n) for the DP array.

✅ Use Cases

  • String segmentation and parsing.
  • Dynamic programming with substring checks.
  • Natural language processing and tokenization.

🧪 Example

Input: s = "leetcode", wordDict = ["leet","code"]
Output: true

Explanation:
- "leetcode" can be segmented as "leet code".

💡 Pro Tips

  • Use a HashSet for fast dictionary lookups.
  • DP state: dp[i] means s[0..i-1] can be segmented.
  • For all possible segmentations, see Word Break II.

The Longest Palindrome problem (LeetCode 409) asks you to find the length of the longest palindrome that can be built with the letters of a given string.

ğŸ¯ Solution Pattern (HashSet for Pair Counting)

// Java Solution
class Solution {
    public int longestPalindrome(String s) {
        Set<Character> set = new HashSet<>();
        int pairs = 0;
        for (char c : s.toCharArray()) {
            if (set.contains(c)) {
                pairs += 2;
                set.remove(c);
            } else {
                set.add(c);
            }
        }
        return set.isEmpty() ? pairs : pairs + 1;
    }
}

🧩 Longest Palindrome – Java Cheat Sheet

📌 What Is It?

The Longest Palindrome problem (LeetCode 409) asks you to find the length of the longest palindrome that can be built with the letters of a given string.

🎯 Solution Pattern (HashSet for Pair Counting)

// Java Solution
class Solution {
    public int longestPalindrome(String s) {
        Set<Character> set = new HashSet<>();
        int pairs = 0;
        for (char c : s.toCharArray()) {
            if (set.contains(c)) {
                pairs += 2;
                set.remove(c);
            } else {
                set.add(c);
            }
        }
        return set.isEmpty() ? pairs : pairs + 1;
    }
}

✅ Use Cases

  • Building the longest palindrome from a set of characters
  • String manipulation and frequency analysis
  • Interview questions on palindromes and hash sets

📝 Algorithm Steps

  1. Iterate through the string, track characters in a HashSet
  2. For each pair, add 2 to the result and remove from set
  3. If any characters remain, you can add one to the center

⏱️ Time Complexity

  • Time: O(N) where N is the length of the string
  • Space: O(K) where K is the number of unique characters

⚠️ Edge Cases to Consider

  • All characters are unique (can use only one)
  • All characters are paired
  • Empty string (should return 0)

🧪 Example

Input: s = "abccccdd"
Output: 7
Explanation: One longest palindrome is "dccaccd", length 7.

Input: s = "a"
Output: 1

🔄 Isomorphic Strings – Java Cheat Sheet

📌 What Is It?

The Isomorphic Strings problem (LeetCode 205) asks you to determine if two strings s and t are isomorphic. Two strings are isomorphic if the characters in s can be replaced to get t, with each character mapping to another character uniquely.

🎯 Solution Pattern (HashMap for Character Mapping)

// Java Solution
class Solution {
    public boolean isIsomorphic(String s, String t) {
        if (s.length() != t.length()) return false;
        Map<Character, Character> mapST = new HashMap<>();
        Map<Character, Character> mapTS = new HashMap<>();
        for (int i = 0; i < s.length(); i++) {
            char c1 = s.charAt(i), c2 = t.charAt(i);
            if ((mapST.containsKey(c1) && mapST.get(c1) != c2) ||
                (mapTS.containsKey(c2) && mapTS.get(c2) != c1)) {
                return false;
            }
            mapST.put(c1, c2);
            mapTS.put(c2, c1);
        }
        return true;
    }
}

✅ Use Cases

  • Checking if two strings have a one-to-one character mapping
  • Pattern matching and transformation
  • Problems involving unique character substitutions

📝 Algorithm Steps

  1. If the strings have different lengths, return false
  2. Use two hash maps to track mappings from s to t and t to s
  3. For each character, check if the mapping is consistent in both directions
  4. If any inconsistency is found, return false
  5. If all mappings are consistent, return true

⏱️ Time Complexity

  • Time: O(n) where n is the length of the strings
  • Space: O(1) (since the alphabet size is fixed, otherwise O(n))

⚠️ Edge Cases to Consider

  • Strings of different lengths
  • Repeated characters mapping to different characters
  • Empty strings (should return true)

🧪 Example

Input: s = "egg", t = "add"
Output: true

Input: s = "foo", t = "bar"
Output: false

Input: s = "paper", t = "title"
Output: true

The Middle of the Linked List problem (LeetCode 876) asks you to return the middle node of a singly linked list. If there are two middle nodes, return the second one.

ğŸ¯ Solution Pattern (Fast and Slow Pointer)

// Java Solution (Fast and Slow Pointer)
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}

ğŸ“ Algorithm Steps

  1. Initialize two pointers, slow and fast, at the head
  2. Move slow by one step and fast by two steps in each iteration
  3. When fast reaches the end, slow will be at the middle
  4. Return slow

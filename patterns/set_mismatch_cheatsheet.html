The Set Mismatch problem (LeetCode 645) asks you to find the duplicate and missing numbers in an array containing numbers from 1 to n, where one number is duplicated and one is missing.

ğŸ§± Pattern Template (Cyclic Sort Solution)

class Solution {
    public int[] findErrorNums(int[] nums) {
        int i = 0;
        while (i < nums.length) {
            int correct = nums[i] - 1;
            if (nums[i] != nums[correct]) {
                int temp = nums[i];
                nums[i] = nums[correct];
                nums[correct] = temp;
            } else {
                i++;
            }
        }
        for (i = 0; i < nums.length; i++) {
            if (nums[i] != i + 1) {
                return new int[]{nums[i], i + 1};
            }
        }
        return new int[]{-1, -1}; // Should not happen
    }
}

ğŸ’¡ Pro Tips

  • Cyclic sort is ideal for in-place detection of duplicates and missing numbers.
  • After sorting, the index where nums[i] != i+1 gives the duplicate (nums[i]) and missing (i+1).
  • Can also be solved with extra space using a HashSet or frequency array.

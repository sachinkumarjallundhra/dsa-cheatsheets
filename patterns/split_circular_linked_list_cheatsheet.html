🔗 Split a Circular Linked List – Java Cheat Sheet

📌 What Is It?

The Split a Circular Linked List into Two Halves problem (LeetCode 725, GFG, etc.) asks you to split a circular singly linked list into two halves. If the number of nodes is odd, the extra node goes in the first list. Both halves should remain circular.

🎯 Solution Pattern (Fast and Slow Pointer)

// Java Solution (Fast and Slow Pointer)
public void splitList(Node head, Node[] result) {
    if (head == null) return;
    Node slow = head, fast = head;
    // Find the middle using fast and slow pointers
    while (fast.next != head && fast.next.next != head) {
        slow = slow.next;
        fast = fast.next.next;
    }
    // For even number of nodes, move fast to the last node
    if (fast.next.next == head) fast = fast.next;

    // Set the heads of the two halves
    Node head1 = head;
    Node head2 = slow.next;

    // Make the first half circular
    slow.next = head1;
    // Make the second half circular
    fast.next = head2;

    result[0] = head1;
    result[1] = head2;
}

✅ Use Cases

  • Splitting a circular linked list into two halves
  • Interview questions on pointer manipulation and circular lists
  • Understanding fast and slow pointer technique

📝 Algorithm Steps

  1. Use fast and slow pointers to find the middle of the circular list
  2. Set the next of the slow pointer to the head to close the first half
  3. Set the next of the fast pointer to the node after slow to close the second half
  4. Return the two heads

⏱️ Time & Space Complexity

  • Time: O(N) for N nodes
  • Space: O(1) (in-place)

🧪 Example

Input: head = [1,2,3,4,5] (circular)
Output: [1,2,3] (circular), [4,5] (circular)

Input: head = [1,2,3,4] (circular)
Output: [1,2] (circular), [3,4] (circular)

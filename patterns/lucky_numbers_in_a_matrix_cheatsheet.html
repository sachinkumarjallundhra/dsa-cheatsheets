🍀 Lucky Numbers in a Matrix – Java Cheat Sheet

📌 What Is It?

The Lucky Numbers in a Matrix problem (LeetCode 1380) asks you to find all lucky numbers in a matrix. A lucky number is an element that is the minimum in its row and the maximum in its column.

🎯 Solution Pattern (Row Min & Column Max)

class Solution {
    public List<Integer> luckyNumbers (int[][] matrix) {
        List<Integer> result = new ArrayList<>();
        int m = matrix.length, n = matrix[0].length;
        
        // Find min in each row
        for (int i = 0; i < m; i++) {
            int minVal = Integer.MAX_VALUE, colIdx = -1;
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] < minVal) {
                    minVal = matrix[i][j];
                    colIdx = j;
                }
            }
            // Check if minVal is the max in its column
            boolean isMax = true;
            for (int k = 0; k < m; k++) {
                if (matrix[k][colIdx] > minVal) {
                    isMax = false;
                    break;
                }
            }
            if (isMax) result.add(minVal);
        }
        return result;
    }
}

✅ Use Cases

  • Matrix analysis and filtering
  • Finding special elements in 2D arrays
  • Data selection based on row/column criteria

📝 Algorithm Steps

  1. For each row, find the minimum value and its column index
  2. Check if this value is the maximum in its column
  3. If yes, add to result list
  4. Return all lucky numbers found

⚠️ Edge Cases to Consider

  • Multiple lucky numbers
  • No lucky numbers
  • Matrix with one row or one column
  • Duplicate values

🧪 Example

Input: matrix = [[3,7,8],[9,11,13],[15,16,17]]
Output: [15]
Explanation: 15 is the minimum in its row and the maximum in its column.

Input: matrix = [[1,10,4,2],[9,3,8,7],[15,16,17,12]]
Output: [12]
Explanation: 12 is the minimum in its row and the maximum in its column.

💡 Pro Tips

  • Always track both row minimum and column maximum
  • Use nested loops for clarity
  • Test with edge cases (single row/column, duplicates)

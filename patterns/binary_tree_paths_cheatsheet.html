🌳 Binary Tree Paths – Java Cheat Sheet

📌 What Is It?

The Binary Tree Paths problem asks you to return all root-to-leaf paths in a binary tree as strings. Each path should be represented as a string with node values separated by "->".

🧱 Pattern Template

class Solution {
    public List<String> binaryTreePaths(TreeNode root) {
        List<String> paths = new ArrayList<>();
        if (root != null) {
            dfs(root, "", paths);
        }
        return paths;
    }

    private void dfs(TreeNode node, String path, List<String> paths) {
        if (node.left == null && node.right == null) {
            paths.add(path + node.val);
            return;
        }
        if (node.left != null) {
            dfs(node.left, path + node.val + "->", paths);
        }
        if (node.right != null) {
            dfs(node.right, path + node.val + "->", paths);
        }
    }
}

📊 Time Complexity

  • Time Complexity: O(N^2), where N is the number of nodes (since each path can be up to N in length).
  • Space Complexity: O(N^2), for storing all paths and recursion stack.

✅ Use Cases

  • Finding all root-to-leaf paths in a binary tree.
  • Tree traversal and path construction problems.
  • Recursive DFS on trees.

📘 Common LeetCode Problems

  • 257. Binary Tree Paths [Easy] DFS

🧪 Example

Input: root = [1,2,3,null,5]
Output: ["1->2->5","1->3"]

Explanation:
- There are two root-to-leaf paths: 1->2->5 and 1->3.

💡 Pro Tips

  • Use DFS to build the path as you traverse the tree.
  • When you reach a leaf node, add the current path to the result list.
  • Edge cases: Empty tree (return empty list), single-node tree.

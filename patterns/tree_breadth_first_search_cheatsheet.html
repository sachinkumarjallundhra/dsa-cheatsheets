üìå What Is It?

The Tree Breadth First Search (BFS) pattern is used to traverse a tree level by level, visiting all nodes at each depth before moving to the next level. It is commonly implemented using a queue.

üéØ Solution Pattern (Level Order Traversal)

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val);
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            result.add(level);
        }
        return result;
    }
}

‚úÖ Use Cases

  • Level order traversal
  • Finding shortest path in unweighted trees
  • Solving problems by levels (e.g., populating next pointers)

üìù Algorithm Steps

  1. Initialize a queue and add the root node
  2. While the queue is not empty:
    • Record the number of nodes at the current level
    • Process each node at this level, adding their children to the queue
    • Store the values for this level
  3. Repeat until all levels are processed

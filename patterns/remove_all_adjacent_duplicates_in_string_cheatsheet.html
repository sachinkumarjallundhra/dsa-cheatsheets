The Remove All Adjacent Duplicates In String problem (LeetCode 1047) asks you to repeatedly remove adjacent duplicate characters in a string until no more can be removed.

ğŸ§± Pattern Template (Stack-Based Solution)

class Solution {
    public String removeDuplicates(String s) {
        Stack<Character> stack = new Stack<>();
        for (char c : s.toCharArray()) {
            if (!stack.isEmpty() && stack.peek() == c) {
                stack.pop();
            } else {
                stack.push(c);
            }
        }
        StringBuilder sb = new StringBuilder();
        for (char c : stack) sb.append(c);
        return sb.toString();
    }
}

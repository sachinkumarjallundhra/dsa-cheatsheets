The Find Center of Star Graph problem (LeetCode 1791) asks you to find the center node of a star graph. A star graph is a tree with one center node connected to all other nodes.

ğŸ¯ Solution Pattern (Edge Analysis)

class Solution {
    public int findCenter(int[][] edges) {
        // The center is the only node that appears in both edges[0] and edges[1]
        if (edges[0][0] == edges[1][0] || edges[0][0] == edges[1][1]) {
            return edges[0][0];
        }
        return edges[0][1];
    }
}
